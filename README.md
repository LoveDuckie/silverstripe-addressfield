# SilverStripe Address field

## Introduction

Address field with geolocation

## Requirements

* SilverStripe CMS ^4.0
* "jeroendesloovere/geolocation-php-api": "1.3.*"

## Installation

```
composer require "thewebmen/silverstripe-addressfield" "dev-master"
```
